# Regression

Regression problems, one self-contained folder each.

- **`car-price-prediction/`** — used-car selling-price regression (CarDekho dataset). Cleaning pipeline, EDA, and a log-target linear baseline (R² ≈ 0.85).

Uses the shared repo-root `.venv` and the "Python (ml)" kernel. See the root `CLAUDE.md` for setup and conventions.
